## Manuel (Chinese)
### 游戏大厅
#### 加入房间
1. 在右侧列表中选择一个房间
2. 点击下方 **加入房间** 按钮

#### 创建房间
1. 在左侧下方 **房间名称** 与 **人数限制** 输入框中输入房间名称与房间最大人数限制  
    - 若 **房间名称** 为空，则默认使用系统随机分配的房间标识符为房间名称
    - 若 **人数限制** 为空或为0，则默认不设置最大人数限制
2. 点击下方 **创建房间** 按钮
